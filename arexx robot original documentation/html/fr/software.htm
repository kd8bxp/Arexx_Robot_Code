AAR ROBOT -CD  Documentations & Logiciels 
Tous les outils logiciels  requis pour le Robot AAR peuvent être trouvés dans cette page. En fonction des réglages de sécurité de votre navigateur Internet, il vous sera possible de commencer l'installation des ces logiciels directement depuis le CD. Toutefois vous serez peut être également obligé de sauvegarder en premier lieu les fichiers dans un répertoire de votre disque-dur afin de les exécuter depuis ce répertoire (effectuez pour ce faire un clic droit de souris sur le lien approprié et sélectionnez l'option "Sauvegarder sous...").

NOTE: Vous devrez être déclaré en tant qu' Administrateur pour pouvoir installer certains logiciels de ce CD !

LOGICIEL ARDUINO  

»Installation

ARDUINO Version 1.0

ARDUINO est une suite logiciel d'outils de développement open source pour les microcontrôleurs AVR d'Atmel™. Celle-ci vous permet de programmer ces microcontrôleurs de façon très simple. Elle est également très simple à installer (il vous suffit de suivre les instructions après avoir exécuté le logiciel d'installation).


Note: Il existe différentes versions pour les ARDUINO™. Nous avons uniquement testé celle présente sur le CD - Les versions plus récentes ou plus anciennes peuvent poser des problèmes. 

»Win2k/XP/Vista/Win7 32 et 64bit
 Driver d'interface USB

Ceci correspond au Driver FTDI USB pour Windows (vous n'avez pas besoin de driver pour Linux du fait qu'il soit déjà intégré dans les nouvelles versions de Kernels de Linux). La nouvelle version CDM 2.06.02 est disponible pour  Win2k/XP/Vista/Win7 32 et 64 Bit. Vous pouvez utiliser CDM_Setup pour  installer le Driver AVANT de raccorder le cordon USB de votre robot pour la première fois ! Si vous désirez installer le driver manuellement, vous trouverez ce dernier dans les dossiers du CD-ROM. La procédure d'installation est décrite dans le Manuel ainsi que dans le Guide d'installation du répertoire du driver.

Attention: Windows 7 et Vista renferment des Drivers standards pour l'interface USB, lesquels ne sont pas compatibles avec le robot AAR. Ils ne pourront donc pas fonctionner avec votre robot ! Il vous faudra IMPERATIVEMENT installer le Driver AVANT de raccorder votre robot au PC pour la première fois ! Si vous ne le faite pas, il vous faudra par la suite mettre à jour manuellement le driver avec le gestionnaire de périphérique de Windows.

Les dernières versions des Drivers peuvent être téléchargées depuis notre site ou depuis le site FTDI: http://www.ftdichip.com/
