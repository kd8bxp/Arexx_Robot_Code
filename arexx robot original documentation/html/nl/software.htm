AAR ROBOT - Documentatie & Software CD
Welkom bij de AAR ROBOT DOCUMENTATIE & SOFTWARE CD.

Hier vind je alle Software die je nodig hebt om te starten met deze nieuwe robot! Je kunt de meeste documenten alleen bekijken als Adobe Reader geïnstalleerd is op de pc. Hier vind je een link naar de website van adobe waar je Adobe reader kunt downloaden. Het hangt af van de veiligheids instellingen en de rechten die je hebt als gebruiker (administrator rechten) of je de software direct kunt installeren vanaf de CD. Misschien moet je wat in de rechten aanpassen of moet je de software eerst naar de harde schijf kopieëren.

LET OP: Je hebt administrator rechten nodig om de software te kunnen installeren!

ARDUINO SOFTWARE

»Install
 ARDUINO Version 1.0

Hier staat de Engelstalige ARDUINO Software

Arduino is een pakket vanexecutabele, open source software ontwikkelings tools voor de Atmel AVR Microcontrollers. Hiermee kun je op een eenvoudige manier Atmel AVR processor programma's schrijven en compileren op een pc. Het programma is eenvoudig te installeren volg de instructies op het scherm nadat de installer gestart is.

LET OP: Er zijn diverse Arduino versies de voorbeeld software op deze CD is alleen getest met deze versie - de nieuwste versie of oudere versies kunne problemen geven met onze voorbeeld software. De software moet dan aangepast worden op de betreffende versie.

»Win2k/XP/Vista/Win7 32 und 64bit
 USB Interface Driver

Deze USB driver heb je alleen nodig voor de Wireless tranceiver als je de AAR draadloos met de RP6v2 wilt gaan besturen.

This is the FTDI USB Driver for Windows (you don't need a Driver for Linux as it is already included in modern Linux Kernels). The newest CDM 2.06.02 Version is available for Win2k/XP/Vista/Win7 32 and 64 Bit. You can use the CDM_Setup to install the Driver BEFORE you attach the USB Interface for the first time! If you want to install it manually, you find the extracted drivers in the apropriate folders on the CD. The Installation is described in the Manual and there are also Installation Guides in the driver folders.

Attention: Windows 7 and Vista can contain a standard Driver for the USB Interface but this one is not complete! It does not have all required Features. So you HAVE TO install the correct Driver BEFORE you attach the USB Interface for the first time! If you did not do this, you have to manually upgrade the Driver with the Windows Device Manager.

De nieuwste versie van de driver kun je direct downloaden van de FTDI website: http://www.ftdichip.com/
