Datenblatt-Verzeichnis

Die folgenden Dokumente richten sich hauptsächlich an fortgeschrittene Benutzer die die Funktionsweise der Hardware der Roboter Arme im Detail verstehen möchten. Aber auch für Einsteiger sind hier nützliche Informationen zu finden - z.B. Pinbelegungen und Erläuterung der Prozessor Register.

Hier können Sie sich das Datenblatt-Verzeichnis ansehen.

Es enthält die folgenden Dateien:
  • ATMEGA328P.pdf - Atmel ATMEGA328P 8-Bit RISC Microcontroller
  • FT232R.pdf - FTDI FT232RL USB UART
  • L293D.pdf - L293D Motorkontroller
  • 74AHC1G14.pdf - Invertierter Schmitt trigger
