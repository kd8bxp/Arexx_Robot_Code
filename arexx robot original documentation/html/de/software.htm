AAR ROBOT - Dokumentations & Software CD
Alle benötigten Software Tools für Ihren AAR Roboter finden Sie auf dieser Seite. Abhängig von den
Sicherheitseinstellungen Ihres Browsers können Sie die Software direkt von dieser CD starten. Eventuell müssen Sie die Dateien zunächst in ein Verzeichnis auf Ihrer Festplatte speichern und diese dann von dort starten (klicken Sie dazu mit der rechten Maustaste auf den entsprechenden Link und selektieren Sie "Ziel Speichern unter" o.ä.).

HINWEIS: Sie müssen sich ggf. vorher als Administrator an Ihrem System anmelden um einige der Softwarepakete installieren zu können!

[TOOL] ADOBE READER 9

WINDOWS SOFTWARE

»Installieren
 ARDUINO Version 1.0

ARDUINO ist eine Sammlung von Open Source Software Entwicklungstools für Atmel AVR Mikrocontroller. Es macht den ATMEL Prozessors unter Windows einfach benutzbar und ist sehr einfach zu installieren - folgen Sie einfach den Anweisungen des Installers.

Hinweis: nur mit diese Versionen von ARDUINO im Software Verzeichnis auf der CD funktionieren alle AREXX Beispielprogramme ohne Problemen. Dies nur falls die neueste Version Probleme verursachen sollte.

»Win2k/XP/Vista/Win7 32 und 64bit
 USB Interface Treiber

Sie benötigen unter Windows diesen Treiber für das USB Interface (in modernen Linux Kerneln ist ein passender Treiber bereits enthalten). Die neueste CDM 2.06.02 Version ist für Win2k/XP/Vista/Win7 32 und 64 Bit verwendbar. ACHTUNG: Unter Windows Vista und Windows 7 ist bereits ein NICHT VOLLSTÄNDIGER standard Treiber vorhanden - dieser wird automatisch installiert wenn Sie das USB Interface das erste mal anschließen. Diesem Treiber fehlen allerdings einige Funktionen! Es ist daher ratsam den Treiber auf dieser CD unbedingt zu installieren BEVOR Sie das USB Interface an den PC anschließen! Wenn Sie dies nicht beachtet haben, müssen Sie über den Gerätemanager den standard Treiber durch den korrekten FTDI Treiber ersetzen.
Sie können den Treiber einfach mit dem CDM_Setup installieren BEVOR Sie das USB Interface das erste Mal anschließen! Wenn Sie den Treiber manuell installieren möchten, finden Sie die Treiberdateien in dem entsprechenden Verzeichnis. Die Installation wird in der Anleitung detailliert beschrieben und den Treibern sind auch Installationsanweisungen beigefügt (allerdings in Englisch).

Neuere Versionen der Treiber können auch direkt von FTDI heruntergeladen werden: http://www.ftdichip.com/
